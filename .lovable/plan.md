

# Site Completo — Cerâmica Livramento

## Identidade Visual
Manter o padrão da landing page atual: **azul escuro + laranja/dourado**, tipografia moderna, logo existente. Design limpo, profissional e 100% responsivo (desktop, tablet e celular).

---

## Páginas do Site

### 1. 🏠 Página Inicial (Home)
- Header com navegação fixa (logo, menu com links para todas as seções)
- Hero impactante com chamada principal e CTA
- Destaques: anos de mercado, 100% paletizado, modelos de telhas
- Seções resumidas de produtos, calculadora, e "nossa história"
- Footer com contato, redes sociais e links rápidos

### 2. 📦 Produtos
- Catálogo dos 4 modelos de telhas (Romaninha, Romana, Portuguesa, Suprema)
- Cada modelo com foto, versões (Natural e Resinada), especificações técnicas (telhas/m², dimensões)
- Seção sobre paletização com fotos do processo e benefícios
- Imagens da landing page atual + placeholders para novas fotos que você enviar depois

### 3. 🧮 Calculadora de Telhas
- Formulário avançado: largura e comprimento do telhado, inclinação, modelo da telha, beiral
- Cálculo automático de quantidade de telhas necessárias
- Resultado com opção de "Solicitar Cotação" já preenchendo os dados
- Visual intuitivo com ilustração/ícones explicativos

### 4. 📋 Solicitar Cotação
- Formulário completo: dados do cliente (nome, WhatsApp, cidade, estado), modelo de telha, tipo (natural/resinada), quantidade estimada
- Integração com banco de dados para salvar todas as cotações
- Ao enviar, o cliente recebe confirmação na tela

### 5. 📖 Nossa História
- Timeline desde 1978 até hoje
- Valores da empresa, missão, números (48+ anos, 500+ parceiros, atuação Norte/Nordeste)
- Fotos da fábrica e equipe (placeholders inicialmente)

### 6. 🤝 Seja Parceiro
- Formulário de parceria (aproveitando o que já existe na landing page)
- Benefícios de ser revendedor

---

## Funcionalidades Especiais

### 💬 Chat com IA + WhatsApp
- **Botão flutuante** no canto da tela com duas opções: Chat no site ou WhatsApp
- **Chat com IA** integrado ao site usando Lovable AI: responde dúvidas sobre produtos, preços, localização de revendedores, e coleta informações do cliente
- **WhatsApp** como fallback para contato direto com a equipe
- O chat com IA pode direcionar o cliente para cotação ou parceria

### 🏪 Encontrar Revendedor (via Chat)
- Integrado ao chatbot: o cliente pergunta sobre revendedores e o chat coleta localização e necessidade
- Os dados ficam salvos no painel admin para a equipe entrar em contato

---

## Painel Administrativo

### 📊 Dashboard de Cotações
- Lista de todas as cotações recebidas com filtros (data, estado, modelo, status)
- Status de cada cotação: **Nova → Em negociação → Fechada → Perdida**
- Botão para enviar resposta via WhatsApp (abre WhatsApp com mensagem pré-formatada com dados da cotação)
- Visualização dos dados do cliente e detalhes do pedido

### 👥 Gestão de Leads do Chat
- Registro de conversas/contatos vindos do chatbot
- Dados de quem procurou revendedor (localização, quantidade de telhas)

---

## Backend (Lovable Cloud)
- **Banco de dados** para cotações, leads do chat, e dados de parceiros
- **Autenticação** para acesso ao painel admin (login da equipe comercial)
- **Edge function** para o chatbot com IA
- **Armazenamento** para fotos de produtos

---

## O que vou precisar de você
1. **Fotos** dos produtos e da fábrica (pode enviar pelo chat a qualquer momento)
2. **Número do WhatsApp** da empresa para integração
3. **Especificações técnicas** de cada telha (telhas/m², peso, dimensões) — se tiver além do que está na landing page
4. **Login/senha** desejados para o painel admin (ou e-mails da equipe)
5. **Texto da história** da empresa (ou posso redigir com base no que já tem)
